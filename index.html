Frequently Asked Questions

Q1: How do I start a new game?
A: To start a new game, simply open the app, select the "Create Game" button, and invite friends to join using the unique game code.

Q2: How do I contact support?
A: You can reach us by emailing fadymorcos@gmail.com for any issues you encounter.

Troubleshooting

If you encounter issues with the app, please try the following steps:
1. Ensure that you have the latest version of the app.
2. Restart the app and try again.
3. If the issue persists, contact us for support.
